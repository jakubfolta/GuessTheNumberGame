# Guess The Number Game
> Small program in which player has to guess the number.

## General info
You have 6 chances to guess the random number. With every guess you will get a small hint.

## Technologies
* Python - version 3.7

## Features
* Have fun by guessing ;)

## Status
Project is finished.
